    About This Project

    Submitted by: Ravit Cohen & Natalie Mirelashvili

    Template used: Basic Game Development Template with HTML5 Canvas and Vanilla JavaScript

    jQuery plugin used: No (All functionality was implemented using pure JavaScript)

    Game Features Implemented:

    • Timer showing remaining game time during gameplay.
    • Spaceship fires special blue bullets (different shot styling).
    • Additional penalties: losing lives when hit by enemy bullets.
    • Enemy ships increase speed and aggressiveness over time.
    • Win condition based on defeating all enemies or scoring over 100 points.
